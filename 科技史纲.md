## 文艺复兴

在意大利的商业贸易中心佛罗伦萨，最早兴起了一弘扬人本主义（或称人文主义）为核心的文艺复兴运动。

人本主义可以说是经院哲学的反面。它歌颂人性，反对神性；提倡人权，反对神权；提倡个性自由，反对宗教桎梏；赞颂世俗生活，反对来世观念和禁欲主义。

## 天文学

爱因斯坦对开普勒成就的评价：知识不能单从经验中得出，而只能从理智的发明同观察到的事实两者的比较中得出。

| 时间 | 国家 | 人物 | 事迹 | 
| -------- | -------- | -------- | -------- | 
| 1473 - 1543| 波兰 | 哥白尼 | 1539 作《天球运行论》（因发现托勒密的天空模型，和观测结果有差异） |
| 1548 - 1600 | 意大利 | 布鲁诺 | 提出无限宇宙论，宣扬哥白尼的日心说，被天主教火刑处死 |
| 1564 - 1601 | 丹麦 | 第谷 | 利用当时最先进的观测技术，广泛、系统、细致、精确地观测并记录天象，将观测结果留给了助手开普勒，告诫曰：一定要尊重观测事实 |
| 1571 - 1630 | 德国 | 开普勒 | 抛弃了正圆运动的概念。 1609年，发表《新天文学》，阐述了开普勒第一定律（火星划出一个以太阳为焦点的椭圆）、开普勒第二定律（由太阳到火星的矢经在相等的时间内划出相等的面积），1618年出版《哥白尼天文学概论》，将以上火星的两个定律推广到太阳系所有行星，同时提出开普勒第三定律（行星公转周期的平方和它同太阳距离的立方成正比）|

## 物理学

| -------- | -------- | -------- | -------- | 
| 1564 - 1642 | 意大利 | 伽利略 | 创造了科学方法：**实验以追究事物之量的数学关系**。发现落体定律（斜面上物的下落距离同所用时间的平方成正比）。建立匀速运动、匀加速运动的概念。改进望远镜，发现月亮上的山脉和火山口、木星的四颗卫星、太阳黑子。|

## 生命科学

关键词：人体结构、血液循环、显微生物学

- 维萨留斯 比利时 1543年出版《人体结构》论述骨骼系统、肌肉系统、血液循环、神经系统、消化系统、内脏系统、脑感觉器官，介绍活体解剖方法。纠正盖伦人体学说和亚里士多德许多错误观点。
- 赛尔维特 西班牙 1511 - 1553 发现血液的肺循环（小循环）（血液由肺动脉进入肺静脉，与这里的空气相混合后流入左心室）
- 法布里修斯 意大利 1603出版《论静脉瓣膜》描述了静脉内壁的小瓣膜，它永远朝着心脏的方向打开，而向相反的方向关闭
- 哈维 创立血液循环理论，1616年公布；1628出版《心血运动论》系统总结了他所发现的血液循环运动规律及其实验依据，是为生理学史上划时代的巨著
- 马尔比基 意大利 1660 用显微镜发现青蛙肺中的血管网，毛细血管
- 列文虎克 荷兰 1632 - 1723 制造了放大倍率高的显微镜；观察到原生生物、血红球、细菌
- 胡克 制造了复式显微镜，发表《显微图》，展现昆虫器官的精细图案
- 斯旺麦丹 荷兰 1637-1682 研究3000多种昆虫的解剖结构

## 20世纪

与19世纪一样，20世纪的重大科学成就，很快就转变成相应的技术，在经济生产和社会生活中发挥作用。但与19世纪不同的是，20世纪的科学更加高深、更加远离我们的日常生活经验，相应地，它所转化的技术实际威力更大、也更难被人类所控制。

最突出的科技中，核能的利用、航空航天技术显示了人类空前的力量，电子信息技术对人类社会属性的再造，医药生物技术对人类自然属性的再造。

### 四大理论模型

- 无时间性图景（以量子力学为代表）

- 有时间性图景
	- 高熵发展（以热力学第二定律为代表）
	- 低熵发展
		- 可预测的（以相对论宇宙学位代表）
		- 不可预测的（以生物进化论为代表）


时间性：

- 热力学第二定律
- 以相对论为基础的现代宇宙学给出了宇宙的整体膨胀图景，而膨胀就意味着宇宙的整体演化，同时也给出了演化的方向性
- 测量问题的不可回避反映了量子力学的不完备性、非封闭性，反映了量子力学以可逆性动力学方程为主干的经典框架的局限性。测量问题提醒着时间之失。
- 耗散结构理论

整体论：

- 量子力学在微观领域，巩固了整体论的基础地位
- 系统论
- 生态学
